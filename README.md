# 호랑봇 안내소 — 사용 설명서

## 파일 구성

```
chungu-site/
├─ index.html        홈 (관리자 진입도 여기)
├─ commands.html     명령어
├─ members.html      멤버 소개 (자소서)
├─ status.html       매칭 · 외출 현황
└─ assets/
   ├─ config.js      ← 관리자 키, 구글 시트 주소
   ├─ style.css      ← 꾸미는 사람은 이 파일만 만지면 됩니다
   ├─ store.js       데이터 저장·불러오기
   └─ app.js         관리자 판별, 공통 헤더
```

## 두 개의 링크

| 대상 | 주소 | 할 수 있는 것 |
|---|---|---|
| 일반 멤버 | `https://주소/` | 명령어·멤버·현황 **보기만** |
| 관리자 | `https://주소/?key=chungu-2026` | 세 페이지 모두 추가·수정·삭제 |

- 키는 `assets/config.js`의 `ADMIN_KEY`에서 바꿉니다. **꼭 바꿔주세요.**
- 관리자 상태는 탭을 닫으면 풀립니다. 홈의 `나가기` 버튼으로도 풀 수 있습니다.
- 관리자로 들어가면 홈에 **일반 공유 주소 복사** 버튼이 나옵니다. 그걸 방에 올리면 됩니다.

## 구글 시트 연동

1. 시트에서 `파일 → 공유 → 웹에 게시 → 쉼표로 구분된 값(.csv)` 선택
2. 나온 주소를 `assets/config.js`의 `SHEETS`에 붙여넣기
3. 각 페이지에 **시트에서 불러오기** 버튼이 생깁니다 (관리자만 보임)

시트 첫 줄은 제목 줄이고, 열 순서는 아래와 같아야 합니다.

- **명령어**: 명령어 / 설명 / 분류 / 관리자전용(Y·N)
- **멤버**: 닉네임 / 나이 / 키 / 전공 / 쉬는요일 / 취미 / MBTI / 매력 / 이상형 / 흡연·주량 / 하고싶은말
- **현황**: 닉네임 / 상태(매칭·외출) / 상대 / 복귀예정 / 메모

`SHEET_LINKS`에 시트 편집 주소를 넣으면 **시트 열기** 버튼도 생깁니다.

## 데이터가 저장되는 곳

브라우저 localStorage입니다. 즉 **수정한 사람의 브라우저에만 남습니다.**
운영자가 여러 명이거나 기기를 바꿀 때는 두 가지 중 하나를 쓰세요.

- 구글 시트를 원본으로 쓰고, 사이트는 불러오기만 하기 (추천)
- 홈의 `데이터 내보내기`로 JSON 저장 → 다른 기기에서 `데이터 가져오기`

## 보안

관리자 키는 브라우저에서 확인할 수 있는 **간이 잠금**입니다. 오픈채팅방 운영용으로는 충분하지만, 남에게 보이면 안 되는 개인정보를 올릴 거라면 서버가 필요합니다. 그때 알려주시면 로그인·DB 붙인 버전으로 확장해 드리겠습니다.

## 꾸미는 사람에게

- 색·글꼴·모서리 둥글기는 전부 `style.css` 맨 위 `:root`에 있습니다. 그 값만 바꿔도 분위기가 완전히 바뀝니다.
- HTML은 건드리지 않아도 됩니다. 클래스 이름만 보고 스타일을 덮어쓰면 됩니다.
- `data-`로 시작하는 속성은 자바스크립트가 쓰는 것이니 지우지 마세요.
